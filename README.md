# Gene Manager Web Application

A dynamic web-based Gene Manager Application built using HTML, CSS, and JavaScript.

This application allows users to:

- Add gene entries
- Edit gene information
- Delete genes
- Search genes dynamically
- View gene statistics
- Manage DNA sequence data interactively

---

# Features

## Gene Management
- Add new genes
- Edit existing genes
- Delete gene entries
- Dynamic UI updates

## Search Functionality
- Real-time search
- Filter by gene name or organism

## Dashboard Statistics
- Total genes count
- Protein coding genes count
- Organism count

## Responsive UI
- Mobile friendly layout
- Interactive cards
- Modern dashboard interface

---

# Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
